使用帮助

下载、安装过程中的常见问题

❓ 浏览器提示"文件可能有害"，被拦截了怎么办？

个人开发者的程序没有大厂的数字签名，浏览器（尤其是 Edge 和 Chrome）会对不常见的 exe 文件默认拦截，这是正常现象，不代表文件有毒。处理方法：

  1. 在浏览器的下载列表中找到被拦截的文件
  2. 点击「…」或右键，选择「保留」/「仍然保留」

❓ Windows Defender / 杀毒软件报毒怎么办？

本站工具均为本人开发，不含任何恶意代码。部分工具使用 Python 等打包而成，或包含模拟键盘鼠标等自动化功能，容易被杀毒软件误报。你可以：

  1. 核对详情页提供的 SHA256 校验值，确认文件未被篡改（见下一条）
  2. 将文件或所在文件夹添加到杀毒软件的白名单/排除项
  3. 不放心的话，可以把文件上传到 VirusTotal 用几十个引擎交叉检测

❓ 怎么核对文件的 SHA256 校验值？

在文件所在文件夹，按住 Shift 点右键 → 「在此处打开 PowerShell 窗口」，运行：

Get-FileHash 文件名.exe

把输出的一长串字符和详情页上公布的校验值对比，完全一致就说明文件完整、没有被篡改。

❓ 双击打开提示"Windows 已保护你的电脑"（SmartScreen 蓝色弹窗）？

点击弹窗中的「更多信息」，再点击「仍要运行」即可。这同样是因为程序没有商业数字签名，属于正常提示。

❓ 工具支持什么系统？

默认支持 Windows 10 / 11（64 位）。个别工具有特殊要求的，会在详情页的「运行环境」中注明。
